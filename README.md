# ReCODE Medical Landing Page

Landing page for [recodemedical.com](https://recodemedical.com), built with Next.js 16, React 19, and Tailwind CSS 4.

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deployment

Deployed via Docker on Dokploy.

```bash
docker compose up --build
```
